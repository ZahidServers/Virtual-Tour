About Us

This Web App is created by Mohammed Zahid Imtiyaz Wadiwale. It is made in Django for Submission of Harvard Cs50 Web Capstone Project. It requires Django 2.0.0 and pillow version 5.0.0. It uses Djangos Built in Template System to display Blogs and uses sqlite3 for dbms models including post and comments. Posts are made using Djangos Admin Portal comments are also moderated from there each comment need to get permission of staff or superuser from admin panel before being visible. Along with post images can also be uploaded. Along with post virtual Tour are also given using a popular javascript library known as three.js a nd custom javascript coding a skybox is rendered out of Panorma Images. Panorma Images are also uploaded from Admin Portal. Also it uses Django Default Pagination sytem to limit no of articles per page by 3 on home page.
